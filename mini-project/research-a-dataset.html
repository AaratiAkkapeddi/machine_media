Part I: Research a dataset

Search through kaggle (you need to make an account to view this page) and choose an image dataset to look through.

Find the following information and submit to the #assignments discord channel:
  • The name of this dataset
  • An example image from this dataset
  • The total number of images in the dataset
  • In what ways are the images in the dataset visually similar? For example, if they are images of faces, are the faces cropped in exactly the same way?
  • In what ways are the images in the dataset visually different? For example, if they are images of apples, do the apples vary in color?
  • What do you think this dataset could be used for? Can you figure out what it is used for?

Instructions for submission:

  • Submit your found dataset information in the #assignments channel.
